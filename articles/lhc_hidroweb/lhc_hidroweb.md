# LHC Hidroweb
-------------------------

**Project description:** A JupyterLab and Python application able to download Rain and Flow through National Water Agency's API. A key tool to use this application is the Leaflet map. The full tutorial to use the tool jump to [Github LHC_Hidroweb](https://github.com/alexnaoki/LHC_Hidroweb).

The text below is a summary of the tool. The ideia is that you'll have to install it correctly just once, than hydrological data should not be a problem.

## 1. Visualization of Inventory

Some metadata is necessary to access and visualize the API. But you can apply your own Inventory or download it from the Water National Agency (You should just use it the Inventory data from the Water National Agency).

<img src="../../images/lhc_hidroweb/inventario01_v2.gif"/>

## 2. Data Selection and Download

In order to acquire **public** data, you should be able to easily download it with minimum* difficulty. After complete the process necessary to use the tool, you'll be able select through:

- **Shapefile**
- **Drawing on the map build**
- **Select by Date**
